/* 始めに */
bbMLpe(bbMemLimiter parameter editor)は、
メモリ最適化プラグインbbMemLimiterのパラメータを変更するためのものです。
設定ファイルによって値を設定できる改造版もありますが、
メモリ最適化のためのプラグインという性質上、極力シンプルな構成にしたかったので
このようなものを作った次第です。直接バイナリを編集しパラメータを変更します。

/* 使い方 */
bbMemLimiter.dllとbbMLpe.exeを同ディレクトリ置き、
bbMemLimterをUnloadした状態でbbMLpeを起動してください。

/* 最後に */
bbMLpe(bbMemLimiter parameter editor)はフリーウェアです。
このソフトウェアによって起こったいかなる損害に関して、
一切の責任を負いませんのでご了承下さい。
